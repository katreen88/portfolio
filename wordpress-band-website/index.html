Project Overview

This project was created using WordPress and the Astra Theme to build a modern and responsive website for a local acoustic band.

The website includes pages for upcoming gigs, embedded videos, contact forms, social media links, and responsive layouts.

Technologies Used

  • WordPress
  • Astra Theme
  • Contact Form 7
  • HTML/CSS
  • Bootstrap
