About Me

I am a passionate and versatile front-end developer with an Associate Degree in Computer Science (58 credit hours). My core strengths lie in building responsive and accessible user interfaces using HTML, CSS, JavaScript, and Bootstrap. Alongside front-end technologies, I have solid experience in object-oriented programming (OOP) with both Python and PHP, and I’m also proficient in PowerShell scripting for automation tasks. My knowledge of databases includes SQL, Microsoft Access, and data manipulation using Excel, allowing me to handle both the interface and logic behind web systems. In addition to programming, I have a creative side — I’m skilled in Adobe Photoshop and Illustrator, which helps me design visually appealing websites and graphics that align with user needs and modern aesthetics. I am currently expanding my development toolkit by learning Git for version control and diving into modern JavaScript frameworks, especially React, to build dynamic and scalable web applications.
